# Recommender System
- [cosine_similarity_algorithm](cosine_similarity_algorithm) contains the recommender system that uses adjusted cosine similarity.
  - This system is trained on [100K item ratings](cosine_similarity_algorithm/test_100k_withtratings.csv) and tested on [100K items](cosine_similarity_algorithm/test_100k_withoutratings.csv).
- [matrix_factorisation_algorithm](matrix_factorisation_algorithm) contains the recommender system that uses matrix factorisation.
  - This system is trained on [10M item ratings](matrix_factorisation_algorithm/test_20M_withratings.csv) and tested on [10M items](matrix_factorisation_algorithm/test_20M_withoutratings.csv).
